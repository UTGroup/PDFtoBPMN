# Жизненный цикл договора ПАО «Авиакомпания «ЮТэйр»

## Описание элементов процесса

Данный документ содержит детальное описание условий выполнения каждого элемента процесса жизненного цикла договора согласно РД-Б7.004-05 «Руководство по договорной работе».

---

## Таблица 1. События процесса

| Название элемента | Тип | Описание и условия выполнения |
|-------------------|-----|-------------------------------|
| **Инициирование договора** | Start Event | **Описание:** Принятие решения о необходимости установления договорных отношений. РПФМ определяет необходимость заключения договора для обеспечения хозяйственной деятельности компании. |
| **Процесс завершен** | End Event | **ЗАВЕРШЕНИЕ (п.18.4-18.6):** Жизненный цикл договора завершен. Договор переведен в архив. Все обязательства сторон по договору прекращены по основаниям, предусмотренным законодательством или договором. Документы сохранены в архиве для обеспечения юридической защиты интересов Авиакомпании и соблюдения установленных законодательством сроков хранения документации. Оригиналы договоров и договорной переписки хранятся в УДО (с момента подписания, но не более 3 лет) и в архиве (фонд действующих договоров - от 3 лет до окончания срока действия договора, далее в архиве согласно номенклатуре дел по РИ-М1.003). Процесс полностью соответствует требованиям РД-Б7.004-05 «Руководство по договорной работе». |
